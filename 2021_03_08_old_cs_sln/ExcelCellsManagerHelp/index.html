Excel Cells Manager ヘルプファイル

Excel Cells Manager Ver1.0.0.0

最終更新日 2021/10/06

概要

エクセルの任意のセルの位置(アドレス)を登録し、登録した位置を表示するアプリケーションです。

機能

  • エクセルの任意のセルの位置(アドレス)を登録します。
  • 登録した位置を表示(セルを選択状態に)します。 表示するワークブックが開いていないときは自動的にワークブックを開きます。
  • アプリケーションが前面にないとき、表示しているスクリーン淵の設定された位置へマウスポインタを移動することで、アプリケーションを前面に表示します。
  • 保存・読込：登録アドレスリストをtsvファイルとして保存、保存したtsvファイルを読み込み表示します。
  • アプリケーションフォーム上にドラッグ＆ドロップすることでファイルを読み込みます。
  • DataGridView の RowHeader ダブルクリックで登録した位置を表示します。
  • 動作時のログ保存、出力をします。
  • CheckedListBox で選択されているワークブックを閉じます。
  • アプリケーション設定をファイル(iniファイル)に保存します。
  • 実行されているエクセルのプロセス(EXCEL.EXE)および開いているワークブックを取得・表示します。 また、アプリケーション実行中は、ワークブックの実行・終了を検知し、自動的に更新します。

ショートカットキー

  • Ctrl＋O：開く
  • Ctrl＋S：ファイルを保存(上書き保存、ファイル名が設定されていないときは名前を付けて保存)
  • Ctrl＋Shift＋S：名前を付けて保存
  • Ctrl＋Shift＋S：名前を付けて保存
  • Ctrl＋Shift＋O：設定ウィンドウを開く

基本的な使い方

  • ボタン[アドレスを登録]：アプリケーションフォーム下部に表示されるアドレス(例：ActiveCell: A1 < Sheet1 < Workbook.xlsm)を、フォーム内の DataGridView に追加します。
  • ボタン[選択する]：フォーム内の DataGridView に表示されているデータのワークブックウィンドウとワークシートをアクティブにし、セルのアドレスを選択します。
  • ボタン[ActiveCell表示]：現在の選択されているセルのワークブックウィンドウとワークシートをアクティブにし、セルのアドレスを選択します。
  • ボタン[コピー]：フォーム内の DataGridView に表示されているデータのワークブックウィンドウとワークシートをアクティブにし、セルの値をクリップボードにコピーします。
  • ボタン[ブックを閉じる]：アプリケーション上部の CheckedListBox に表示されているワークブックがチェック状態のとき、そのワークブックを閉じます。
  • ボタン[リスト更新]：現在開いているエクセルのプロセス(EXCEL.EXE)をすべて取得し、さらにそのプロセスが保持しているワークブックを取得したのち、その一覧をアプリケーション上部の CheckedListBox に表示します。

動作確認環境

OS Version : Windows10 20H2 (OS ビルド 19042.1237)
.NET Framework Version : .NET Framework 4.8
